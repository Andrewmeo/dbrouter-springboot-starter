# dbrouter-springboot-starter

该分库分表路由组件基于 HashMap 扰动函数实现路由策略

## 1 软件架构
SpringBoot


## 2 安装教程

1.  mvn install


## 3 使用说明

1.  打包后在项目中引入pom
```java
    <dependency>
        <groupId>com.xhystudy.dbrouter</groupId>
        <artifactId>db-router-spring-boot-starter</artifactId>
        <version>1.0.0-SNAPSHOT</version>
    </dependency>
```
